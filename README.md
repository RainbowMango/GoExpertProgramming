# 《GO专家编程》

![](cover.png)

## 图书介绍
本书涵盖内容：
- 常见数据结构、控制结构的实现原理剖析；
- 常用标准库实现原理剖析；
- 工具链使用、原理剖析（如 Go test、Go Module等）；
- 编程陷阱及相关真实案例解析；
